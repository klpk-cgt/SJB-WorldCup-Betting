# 2026 美加墨世界杯朋友群观赛竞猜站 + AI 智能助手

这是一个专为朋友聚会、微信看球群设计的 2026 世界杯娱乐积分竞猜及 AI 观盘互动全栈网页系统。本系统完全由虚拟娱乐积分驱动，旨在活跃看球气氛。**不支持任何真实资金充值、提现或转换。**

## 🌟 核心版画功能

- 🏟️ **比赛信息中心**：查看 104 场 2026 世界杯焦点日期、实时赛赛事态（进行中、FT 已完赛、NS 未开赛）、进球球员、战术首发阵容。
- 🔮 **Polymarket 风格竞猜**：涵盖 “90分钟胜平负”（含加时及点球说明）、“精确比分波胆” 及 “总进球数大小 2.5分” 娱乐玩法。
- 📊 **合规风控模块**：限制单赛事最高投投额不得多于总身家的 50%，最低需留 100 积分防洗牌。
- 🏆 **群友排行榜**：包含 “总积分霸主位榜”、“命中率精锐榜” 及 “单场搏冷奇迹 ROI 奖”，金银铜奖章代表荣耀，AI 动态赋予趣味玩家专属称号。
- 🧠 **Gemini AI 智能大脑**：自动生成每日推荐看点评测，根据即时盘口指数提供有趣的赛前奇门推演。
- ⚙️ **群主后台管理 `/admin`**：人员一键预置分配、比分成绩即时录入、预测自动化结算与派彩、API 同步仿真。

## 🛠️ 技术组合

- **前端**：React 19, Vite, Tailwind CSS v4, Lucide Icons, Mobile-first.
- **后端**：Express ES6 实时控制器, JSON 文件持久化数据库服务（数据修改防丢失）。
- **AI 能力**：Google GenAI Modern SDK (`gemini-3.5-flash` 黄金能效模型）。

## 🚀 极速本地部署与运行

开发环境自带完整全量 SEED 种子数据。

### 1. 安装依赖包
```bash
npm install
```

### 2. 开发者模式极速启动
在 `.env` 或开发环境变量中设置 `GEMINI_API_KEY`（留空则自动降级到精细本地预测模型）。
```bash
npm run dev
```
打开浏览器访问：`http://localhost:3000`

### 3. 世界级生产编译
生产环境中会将全栈 Express + Vite 的 React SPA 全量压缩：
```bash
npm run build
npm start
```
